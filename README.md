# discord-bot

A discord bot that provides Ethereum blockchain data on demand.

## Tools used:
* Python
* APIs

## Current Commands:
* $gas
    * Retrieves current Ethereum gas prices and posts them to discord.
* $eth
    * Retrieves the current price of Ethereum and posts it to discord.
